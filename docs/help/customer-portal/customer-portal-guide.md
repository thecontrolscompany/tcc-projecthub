# Customer Portal Guide

## Overview

The customer portal gives you a read-only view of your TCC projects.

Main route:

- `/customer`

From the portal, you can:

- view active projects you have access to
- open a project for more detail
- review billing and progress
- read weekly updates
- review approved change orders
- review materials / BOM status
- open printable reports
- send project-specific feedback

The portal is for visibility. Customers do not edit project records directly here.

---

## Dashboard

When you first open the portal, you will see your project dashboard.

The dashboard includes:

- summary cards
- a financial snapshot by project
- a billed vs backlog summary
- search and filter controls
- one card for each active project

You can search by project name or address and filter the list by status.

---

## Opening a Project

Select any project card to open the project detail page.

The project page may include:

- project name
- customer name
- site address
- general contractor
- scope description
- scheduled completion date
- photo count
- overall progress
- financial summary cards

If a completion date has not been entered yet, the portal may show a schedule warning.

---

## Financial Information

The project detail page shows a quick financial summary, including:

- contract value
- total billed
- remaining balance
- percent complete

If approved change orders exist, they are included in the contract value shown to the customer.

The page may also show:

- a progress-over-time chart
- a billing summary chart
- a billing history table by month

If no billing data has been entered yet, the portal will show an empty-state message instead of a chart or table.

---

## Approved Change Orders

If the project has approved change orders, they appear in a dedicated section.

That section may show:

- change order number
- title
- approval date
- reference document
- amount

Only approved change orders are shown in the customer portal.

Draft, pending, rejected, or void change orders are not shown.

---

## Weekly Updates

The `Weekly Updates` tab shows submitted weekly updates from the project team.

A weekly update may include:

- week ending date
- percent complete
- notes
- crew activity
- material delivered
- equipment set
- safety incidents
- inspections / tests
- delays / impacts
- other remarks

Each update can include a `View Report` link.

That link opens a printable weekly report page.

If the update includes BOM content, you may also see an `Includes BOM` badge on the update card.

---

## Materials / BOM

The `Materials` tab shows a read-only Bill of Materials view for the project.

You may see:

- material sections
- descriptions
- required quantities
- received quantities
- status such as missing, partial, received, or surplus

The tab also includes a `Generate BOM Report` button.

That button opens a printable BOM report for the project.

---

## Project Team

If team assignments are available, the portal shows a contact section for your project team.

It may include:

- Project Manager
- Field Lead
- Operations Manager

Contact details may include:

- name
- email
- phone number

---

## Public Status Link

Some projects include a public status link.

This is a simplified external-facing project status page that can be shared more broadly.

It shows limited information such as:

- project name
- current percent complete
- last update date
- contact information

It does not show the full financial or internal project detail available inside the main customer portal.

---

## Feedback and Questions

The project detail page includes a feedback panel for project-specific questions or comments.

You can use it to:

- ask for clarification
- report a billing question
- ask about an update
- flag missing or confusing information

Messages submitted there are tied to the current project.

---

## Common Questions

### Why do I only see some projects?

You only see projects where your account has been granted portal access.

### Why can’t I edit project information?

The customer portal is read-only for project records.

### Why don’t I see pending change orders?

Only approved change orders are shown.

### Why is a chart or table empty?

That usually means the related data has not been entered yet.

### Why does the public status page show less information?

It is intentionally limited so it can be shared safely with a broader audience.

---

## Need Help

If you cannot access a project, believe information is missing, or need clarification:

- use the project feedback form in the portal
- contact the listed project team member
- contact The Controls Company directly
