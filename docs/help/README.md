# Help Center

This folder contains end-user help content for the TCC ProjectHub application.

## Structure

- `customer-portal/`
  - Help content for customer-facing users in `/customer` and related customer-visible pages
- `pm-portal/`
  - Reserved for future PM portal help content
- `admin/`
  - Reserved for future admin help content
- `time/`
  - Reserved for future TCC Time help content

## Writing goals

- Keep help content task-oriented and easy to scan
- Explain what users can do today, not future roadmap features
- Match the actual UI labels and page names used in the app
- Separate customer-facing guidance from internal/admin guidance

## Current status

- Customer portal help has been drafted first
- Other sections are placeholders for future guides
