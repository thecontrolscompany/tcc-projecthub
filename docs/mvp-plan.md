# TCC ProjectHub MVP Plan

## Phase 1 — Foundation
- Build the public landing page and role-based portal placeholders
- Establish shared UI patterns and route structure
- Keep room for Supabase auth and data wiring

## Phase 2 — Data + Auth
- Create Supabase project and environment variables
- Add tables for users, customers, projects, weekly updates, and billing periods
- Implement role-based authentication and protected routing

## Phase 3 — PM Workflow
- Assigned project list
- Weekly report submission
- Percent-complete updates and blocker tracking

## Phase 4 — Admin Billing Workflow
- Review active projects by billing period
- Apply Excel-derived billing calculation
- Support roll-forward and email preparation

## Phase 5 — Customer View
- Read-only project update timeline
- Billing history visibility
- Secure customer-specific access

## Future Hooks
- Microsoft Graph / OneDrive document links
- QuickBooks Online sync for contract values and invoice history
